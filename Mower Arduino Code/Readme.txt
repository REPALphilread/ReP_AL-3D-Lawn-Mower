ReP_AL 3D Printed Robot Lawn Mower
----------------------------------

Code V6.01   
------------------------------

- Improved the docking detection by amp sensor
- Improved Serial communication between the NANO and the MEGA.  (Thanks to Adam Kaczmarzyk)
